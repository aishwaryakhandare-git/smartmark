SmartMark — AI Powered Face Recognition Attendance System

SmartMark is an AI-based attendance management system that uses Face Recognition Technology to automate attendance tracking for colleges, classrooms, and institutions. The platform includes dedicated modules for Admin, Teacher, and Student management along with scheduling, attendance analytics, subject management, reminders, and a modern dashboard interface.
